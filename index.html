Instalación de QtCreator

Puedes descargar la instalación directamente de la pagina oficial. Se recomienda descargar la instalación offline, ya que debido al peso de Qt, es probable que si surgen problemas de conexión durante la instalación este quede mal instalado. Si la descarga está muy lenta desde el sitio oficial de Qt, prueba descargar la version que hay disponible en este mirror UTFSM de QtCreator 5.7 x64 Linux

Recuerda que para ejecutar este archivo desde linux, primero debes agregarle permisos de ejecución:

$ chmod +x /path/to/your/file.run

Instalación de OpenCV

  • Clonar la hebra principal del repositorio oficial de OpenCV:
$ cd
$ mkdir -p OpenCV-3.0.0/OpenCV-3.0.0-master
$ cd OpenCV-3.0.0/OpenCV-3.0.0-master
$ git clone https://github.com/Itseez/opencv.git
  • Clonar la hebra "contrib" del repositorio oficial de OpenCV:
$ cd ~/OpenCV-3.0.0
$ mkdir OpenCV-3.0.0-contrib
$ cd OpenCV-3.0.0-contrib
$ git clone https://github.com/Itseez/opencv_contrib.git
  • Compilar el codigo fuente de OpenCV, en conjunto con contrib:
$ cd ~/OpenCV-3.0.0/OpenCV-3.0.0-master/opencv
$ mkdir build && cd build
$ cmake -D CMAKE_BUILD_TYPE=Release -D OPENCV_EXTRA_MODULES_PATH=/home/${USER}/OpenCV-3.0.0/OpenCV-3.0.0-contrib/opencv_contrib/modules -D CMAKE_INSTALL_PREFIX=/usr/local ..
$ make -j$(nproc)
$ sudo make install
$ sudo ldconfig
  • Finalmente para utilizar la biblioteca de OpenCV, agregar la siguiente línea en el archivo .pro del proyecto en QtCreator:
LIBS += `pkg-config --cflags --libs opencv`
